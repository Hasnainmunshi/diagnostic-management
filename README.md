# 🏥 Diagnostic Management System API

A scalable and secure backend API for managing diagnostic services, including user authentication, patient handling, appointments, and report management.

Built with modern best practices using **Node.js, Express, TypeScript, and MongoDB**.

---

## ✨ Overview

The Diagnostic Management System API provides a robust backend for healthcare platforms. It supports role-based access, secure authentication, and modular architecture for easy scalability.

---

## 🚀 Key Features

- 🔐 JWT Authentication (Access + Refresh Token)
- 🛡️ Role-Based Authorization (Admin / User / Doctor)
- 👤 User Management System
- 📅 Appointment Scheduling
- 🧾 Diagnostic Report Handling
- ⚡ RESTful API Design
- 🧩 Modular & Scalable Architecture
- 🛑 Centralized Error Handling
- 🔒 Secure HTTP Headers (Helmet)
- 🌐 CORS Enabled
